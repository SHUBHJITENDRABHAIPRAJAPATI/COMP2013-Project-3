# Grocery Store App Alpha

## Description
Full stack groceries application built with React and Node.js
Users are able to register, log in, view products, manage a cart, and add/edit products if they are an admin.

## Routes
/               - Login page  
/login          - Login page  
/register       - Register page  
/main           - Main grocery app  
/add-product    - Add a new product  
/edit-product/:id - Edit a product  
*                - Page not found or unauthrorized
